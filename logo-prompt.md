# Streifen — Logo Prompt

## Concept

Minimalist macOS app icon for a window manager called "Streifen" (German for stripes/strips).

## Primary Prompt

```
Minimalist macOS app icon, flat design. Three horizontal rectangular strips
side by side, slightly overlapping, representing windows in a horizontal
scroll strip. Colors: deep navy blue (#1a1a2e) background with three strips
in graduated tones — cool white (#f0f0f5), light steel blue (#b8c6db),
and warm silver (#d4d4dc). Subtle rounded corners on each strip. No text.
Clean negative space. Suitable for macOS menu bar and dock at 1024x1024px.
Modern, geometric, premium feel. No gradients, no shadows, no skeuomorphism.
```

## Monochrome Variant (Menu Bar)

```
Monochrome icon, 22x22px template image for macOS menu bar. Three thin
vertical rectangles side by side with small gaps between them, representing
window strips. Simple geometric shapes only. Black on transparent background,
designed as macOS template image (system auto-tints for light/dark mode).
```

## Alternate Directions

1. **Abstract S:** Three horizontal lines forming a stylized "S" shape, each line a different width
2. **Scroll indicator:** Three rectangles with a subtle horizontal arrow suggesting scrollability
3. **Workspace grid:** Small 3x3 dot grid with one row highlighted, representing active workspace
