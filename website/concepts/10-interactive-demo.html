What you just saw

The Horizontal Strip

Every workspace is a horizontal strip. Windows sit side-by-side, never overlapping. Click a window above to focus it — the strip scrolls to center it. This is how Streifen manages your screen: spatially, predictably, keyboard-first.

The strip above is a functional simulation. The real thing manages your actual macOS windows through the Accessibility API.

Try it: press 1-9

Nine Workspaces, Instant Switch

Each number key loads a different workspace with its own set of windows. Your contexts stay separated. Code on 1, communication on 2, research on 3 — you decide the layout.

In the real app, switching is triggered with Hyper+number (Caps Lock remapped as Hyper). Zero animation delay. Your hands never leave the keyboard.

Smart placement

App Behavior Modes

Notice how some windows appear on every workspace while others stay put? Streifen assigns each app a behavior mode that controls how it moves between workspaces.

Pinned
Locked to a specific workspace. Teams stays on 2. Mail stays on 3. Always where you expect it.
Follow
Follows you between workspaces. Finder and Notes are always one shortcut away.
Normal
Stays where you opened it. Standard behavior for most applications.
Under the hood

Native macOS, No Hacks

Streifen is a Swift menu bar app. It reads and positions windows through the macOS Accessibility API. No System Integrity Protection bypass. No code injection. No Electron runtime. Lightweight, fast, and inspectable.

Inspired by PaperWM (GNOME), rebuilt from zero as a first-class macOS citizen.
