Streifen — Dein Desktop, horizontal gedacht.

A horizontal scrolling window manager for macOS. Windows arrange themselves side by side in a continuous strip — no overlapping, no hunting through piles. Inspired by PaperWM for GNOME, built native for Mac.

The Strip

Every workspace is a horizontal strip. Windows line up side by side, like cards on a table. Scroll through them. The focused window stays centered. No stacking, no z-order confusion.

Workspaces 1-9

Nine virtual workspaces. Each with its own strip of windows. Switch with Hyper + number. Your context stays intact — no animation delays, no spatial confusion.

Hyper key = remapped Caps Lock (via Karabiner or similar)

Smart App Placement

Three modes determine where a window appears. Assign once per app, and Streifen remembers.

Pinned — always WS #n Follow — opens where you are Normal — stays where placed
Teams pinned to WS 1. Finder follows. VS Code stays put.

Keyboard Shortcuts

Everything accessible without touching the mouse.

  • Hyper + 1-9 Switch workspace
  • Hyper + ←→ Focus prev / next
  • Hyper+Shift + ←→ Reorder window
  • Hyper + F Full width
  • Hyper + D Half width
  • Hyper+Shift + 1-9 Move to workspace

Resize Presets

Windows snap to predefined widths: full, half, third, two-thirds. No manual dragging. Pick a preset, keep moving. The strip reflows automatically.

Presets are configurable. Default set covers typical screen ratios.

No Hacks Required

Streifen uses the macOS Accessibility API exclusively. No System Integrity Protection modifications needed. No private Apple APIs. Works on any standard macOS install with accessibility permissions granted.

Native macOS

Built with Swift. Runs as a menu bar app — lightweight, unobtrusive. No Electron, no web views, no background processes consuming resources. Just a clean macOS citizen.

Open Source

Streifen is open source. Read the code. Contribute. File issues. Built because macOS deserved a proper scrolling window manager, not another tiling hack.
